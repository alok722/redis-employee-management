> A node-redis webapp to manage employee database


## ❓ what is it?

A simple nodeJs & redis webapp to manage employee database.

## 📜 usage

Make sure you have Redis installed and running

Install Dependencies

```sh
$ npm install
```

Run Server

```sh
$ npm start
```

Visit http://localhost:3000
